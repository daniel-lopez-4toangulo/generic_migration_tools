Custom Currency Rate


This Module will Add functionality of custom currency rate in Sale and Invoice Orders. If the custom rate is defined, it will be considered and the journal entries in the system will be according to this rate.

Define custom rate in Sale order.
The debit and credit in Journal entries will be with the custom rate defined.
